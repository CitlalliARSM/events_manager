@mixin fluid-type();

This mixin resizes any css property fluidly taking 5 arguments:

1.property
2.Finishing in Resolution...
3.Starting in Resolution...
4.Ending measure...
5.Starting measure...


In the next example it is applied so:

@include fluid-type('padding-top',$media-small, $media-big, 100px, 20px );

@include fluid-type('font-size',$media-small, $media-big, 14px, 60px );

@include fluid-type('padding-bottom',$media-small, $media-big, 100px, 20px );

Im a heading resized fluidly by the Awesome mixing above :)
